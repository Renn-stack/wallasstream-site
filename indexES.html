Dos modos de escucha

Elige cómo reacciona Wallas’ Stream cuando el audio se detiene.

Conectado (desconexión automática)

Se desconecta automáticamente después de silencio para que vuelvas a escuchar tu entorno.

Ideal para música, videos y uso diario.

Para más información, consulta Dos modos de escucha

Modo Reunión (mantener conexión)

Permanece conectado durante el silencio para que el audio no se corte.

Ideal para reuniones, clases y pausas largas.

Modo de Escucha Compartida

Ve juntos. Escucha juntos.

  • Perfecto para películas, clases o escuchar en grupo
  • Varios dispositivos pueden conectarse al mismo tiempo
  • Todos escuchan el mismo audio, en tiempo real
  • Sin problemas de emparejamiento ni cambios de Bluetooth

Ejemplo: cuatro dispositivos conectados simultáneamente.

Lo que necesitas

  • ✓ Una computadora compatible
  • ✓ Un dispositivo móvil con Wallas’ Stream instalado
  • ✓ Un dispositivo auditivo conectado a tu dispositivo
  • ✓ Wi-Fi o un cable USB
Cómo funciona
1

Paso 1: Selecciona Wallas Audio

Selecciona Wallas Audio como dispositivo de sonido en tu computadora (entrada y salida).

Puedes cambiarlo cuando quieras.

2

Paso 2: Elige un modo

  • Wi-Fi → Escanea el código QR
  • USB → Conecta un cable USB

Para más información, consulta Dos modos de escucha.

3

Paso 3: Empieza a escuchar

Audio en vivo de tu computadora, transmitido de forma privada a tu dispositivo auditivo.
